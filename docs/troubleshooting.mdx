# Troubleshooting & FAQs

This guide provides solutions to common issues you may encounter while using SPT Teams.

## 🔧 **Common Issues**

### **Authentication & Access**

#### **Can't Log In**
**Solutions**:
1. Check credentials and reset password if needed
2. Clear browser cache and cookies
3. Try different browser or device
4. Check internet connection
5. Contact admin if issue persists

#### **Access Denied Errors**
**Solutions**:
1. Check your user role and permissions
2. Verify workspace access
3. Contact admin to review permissions
4. Check department-level access
5. Verify feature-specific permissions

### **Workspace Management**

#### **Can't Access Workspace**
**Solutions**:

## ❓ **Frequently Asked Questions**

### **General Questions**

#### **Q: How do I reset my password?**
**A**: Go to login page → "Forgot Password" → Enter email → Follow instructions

#### **Q: How do I enable multi-factor authentication?**
**A**: Settings → Security → Multi-Factor Authentication → Follow setup

#### **Q: How do I change my profile information?**
**A**: Profile → Edit Profile → Update information → Save

#### **Q: How do I invite new users?**
**A**: Users → Invite Users → Enter email and details → Set permissions → Send

### **Workspace Questions**

#### **Q: How do I switch between workspaces?**
**A**: Click workspace name in top navigation → Select desired workspace

#### **Q: How do I manage workspace permissions?**
**A**: Settings → User Management → Permissions → Configure permissions

### **Team Questions**

#### **Q: How do I create a new team?**
**A**: Teams → Create Team → Fill details → Add members → Set permissions

#### **Q: How do I add members to a team?**
**A**: Teams → [Team Name] → Members → Add Member → Select users

### **Financial Questions**

#### **Q: How do I submit an expense?**
**A**: Financial → Expense Management → Submit New Expense → Fill details → Submit

#### **Q: How do I create a budget?**
**A**: Financial → Budget Tracking → Create Budget → Configure details

### **HR Questions**

#### **Q: How do I add a new employee?**
**A**: HR → Employee Management → Add Employee → Fill information

#### **Q: How do I request leave?**
**A**: HR → Leave Management → Request Leave → Select type and dates

### **AI Assistant Questions**

#### **Q: How do I get AI recommendations?**
**A**: AI Assistant → Workflow Recommendations → Generate Recommendations

#### **Q: How do I analyze documents with AI?**
**A**: AI Assistant → Document Intelligence → Select document → Analyze
